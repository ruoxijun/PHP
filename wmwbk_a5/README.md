# 无名轻博客PHP版

![输入图片说明](http://demo.semlog.cn/v1.2.jpg "截图_2.jpg")

#### 介绍

无名轻博客v2.0正式版发布
运行环境：PHP>=5.4 pdo_sqlite

#### 更新如下：

- 1.密码保存到数据库中，采用md5双层加密。

- 2.增加博客关键词配置

- 3.增加侧栏开关

- 4.文章增加关键词配置

- 5.增加文章分类

#### 使用前请修改密码


- 1、默认密码：admin
- 2、uptov2.0.php为v1.2升级到v2.0程序，无用可以删除。
